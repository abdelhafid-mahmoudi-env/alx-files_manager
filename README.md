# Files Manager

## Overview
This project is a file management service implemented with Node.js, MongoDB, and Redis. It provides the ability to upload, view, and manage files with user authentication, background processing for image thumbnails, and pagination for file listings.

## Features
- User authentication with JWT tokens
- Upload files and images
- Generate thumbnails for images in the background
- List and retrieve files with pagination
- Publish/unpublish files
- Retrieve file content with correct MIME type
- Basic user management
